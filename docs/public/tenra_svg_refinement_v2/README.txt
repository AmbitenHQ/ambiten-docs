# Tenra SVG Refinement v2

This polish pass includes:
- tighter geometry and fewer decorative details
- light and dark variants for all 14 core icons
- normalized runtime boundary system
- improved small-size readability
- refined logo pack and spirit illustration

## Included
- Logo mark: light + dark
- Wordmark: light + dark
- Primary logo: light + dark
- Spirit illustration
- 14 icons in light
- 14 icons in dark

## Design decisions
- Outline-first system for docs, dashboards, and navigation
- Reduced visual noise vs v1
- Consistent 7–10px stroke scale in 512px masters
- Gradient reserved for emphasis, not overload

## Recommended next step
Move these into a final vector editor pass to:
1. tune optical alignment
2. adjust wordmark kerning precisely
3. create 16/20/24px pixel-fitted exports
4. package a favicon/app icon bundle
